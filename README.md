# Financial Determinants of Capital Gearing: A Regression Analysis

This project performs a multivariate linear regression analysis to identify the key financial drivers behind corporate **Gearing** (leverage). Using data extracted from the **FAME database**, the model evaluates the predictive power of profitability and efficiency ratios on a firm's debt-to-equity structure.

## üìä Regression Summary

The model analyzes 40 observations to determine the relationship between **Gearing** (Dependent Variable) and several independent variables:

* **ROCE** (Return on Capital Employed)
* **ROSF** (Return on Shareholders' Funds)
* **RTA** (Return on Total Assets)
* **Profit Margin**

### Key Statistics

* **R-Square:** 0.262 (Approximately **26.2%** of the variance in Gearing is explained by these four variables).
* **Significance F:** 3% (The overall model is statistically significant at the 95% confidence level).
* **Primary Driver:** **ROSF** is the most significant predictor with a P-value of **1%**.

## üîç Analytical Insights

* **Inverse Relationship:** The coefficient for **ROSF** is **-4.44**, suggesting that as returns to shareholders increase, gearing tends to decrease (aligning with the Pecking Order Theory of finance).
* **Model Accuracy:** The **Standard Error of 140.9** and the R-Square value suggest that while these ratios are influential, there are other external factors not captured in this specific model that influence gearing decisions.
